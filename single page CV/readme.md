# Single Page CV

## Project Overview
This is a **Single Page CV** project built using **pure HTML**, with minimal inline CSS for styling and a little JavaScript for interactivity.  
The project showcases a professional CV layout on a single page, including sections like About Me, Skills, Experience, Education, and Contact. It is fully responsive and works on both desktop and mobile devices.

## Live Project Link
[View My Single Page CV](https://alririn.github.io/Single-Page-CV/)

## Features
- Single page layout with all sections in one page
- Inline CSS used for styling and colors
- JavaScript used to dynamically change the page title
- Responsive and simple design
- Easy to customize

## Technologies Used
- **HTML** – Structure of the CV  
- **Inline CSS** – Styling and colors  
- **JavaScript** – Simple interactivity (e.g., changing `document.title`)
